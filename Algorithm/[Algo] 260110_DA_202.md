### 슬라이딩 윈도우 1

```python
# 마지막 시작 구간은 len(numbers) - window
def solution(numbers, window):
    answer = []

    for i in range(len(numbers) - window + 1):
        win_sum = sum(numbers[i : i + window]) # 슬라이싱 사용해 합계 구하기
        answer.append(win_sum)

    return answer
```
