# 오늘의 알고리즘 (2026.01.23)

## 오늘 푼 문제
- **백준 1546번**: [평균](https://www.acmicpc.net/problem/1546) (Bronze 1)

---

## 내 풀이
```python
# 1. 과목 개수 입력받기
N = int(input())

# 2. 점수들을 리스트로 입력받기
scores = list(map(int, input().split()))

# 3. 최댓값 찾기
M = max(scores)

# 4. 새로운 점수 계산하기
new_scores = []
for s in scores:
    new_score = s/M*100
    new_scores.append(new_score)

# 5. 새로운 평균 구하기
avg = sum(new_scores)/N

# 6. 평균 출력
print(avg)
```

## 새로 배운 개념
### 1. map()과 list()의 조합
- `map(int, input().split())`은 입력받은 문자열을 공백 기준으로 나누고 각각을 정수로 변환합니다.
- 하지만 `map` 객체는 그 자체로 바로 사용하기 어렵기 때문에, `list()`로 감싸서 리스트 형태로 변환해야 값에 접근하고 사용할 수 있다는 점을 배웠습니다.

### 2. 리스트 관련 유용한 함수들
- `max()`: 리스트에서 최댓값을 찾는 함수입니다.
- `sum()`: 리스트의 모든 요소를 더한 값을 반환합니다.
- `append()`: 리스트에 새로운 값을 추가할 때 사용합니다.

### 3. for 반복문을 통한 리스트 변환
- 빈 리스트를 만들고 (`new_scores = []`)
- `for` 반복문으로 기존 리스트의 각 요소를 순회하며
- 계산한 결과를 `append()`로 새 리스트에 추가하는 패턴을 익혔습니다.

## 덧붙임 (Self Feedback)

- 빈 리스트를 만들고 반복문으로 값을 추가하는 패턴이 앞으로 많은 문제에서 유용하게 쓰일 것 같습니다.
- `map()`을 `list()`로 감싸야 한다는 점을 확실히 이해했고, 이런 입력 처리 방식이 알고리즘 문제에서 자주 쓰인다는 것을 알게 되었습니다.
- 수학 공식을 코드로 구현하는 연습이 되었고, 평균 계산처럼 간단해 보이는 문제도 단계별로 접근하는 습관의 중요성을 깨달았습니다.
