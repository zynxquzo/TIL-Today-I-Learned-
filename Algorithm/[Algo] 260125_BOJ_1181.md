# 📚 오늘의 알고리즘 (2026.01.25)

## ✅ 오늘 푼 문제
- **백준 1181번**: [단어 정렬](https://www.acmicpc.net/problem/1181) (Silver 5)

---

## 💻 내 풀이
```python
N = int(input())  # 입력받을 단어의 개수

lst = []  # 단어들을 저장할 빈 리스트 생성
for _ in range(N):  # N번 반복
    words = input()  # 단어 하나씩 입력받기
    lst.append(words)  # 입력받은 단어를 리스트에 추가
    
lst = list(set(lst))  # set()으로 중복 제거 후 다시 list로 변환

lst.sort()  # 사전순으로 정렬 (2순위 - 먼저 정렬)
lst.sort(key = len)  # 길이순으로 정렬 (1순위 - 나중 정렬이 우선순위 높음!)

for word in lst:  # 정렬된 리스트의 각 단어를
    print(word)  # 한 줄씩 출력
```

## 📖 새로 배운 개념
### 1. set()을 이용한 중복 제거
- `set()`은 중복을 자동으로 제거해주는 자료구조입니다.
- 하지만 `set()`은 순서를 유지하지 않기 때문에, 정렬이 필요한 경우 `list(set(lst))`로 다시 리스트로 변환해야 합니다.
- `set`과 `list`의 차이점을 명확히 이해하게 되었습니다.

### 2. 다중 조건 정렬 - 안정 정렬(Stable Sort)의 원리 ⭐
- **핵심 개념**: 나중에 정렬하는 것이 우선순위가 높습니다!
- 파이썬의 `sort()`는 안정 정렬이기 때문에, 정렬 기준이 같은 값들은 이전 순서를 유지합니다.
- 따라서 우선순위가 낮은 조건(사전순)을 먼저 정렬하고, 우선순위가 높은 조건(길이순)을 나중에 정렬해야 합니다.
  - 1단계: `lst.sort()` - 사전순 정렬 (2순위)
  - 2단계: `lst.sort(key=len)` - 길이순 정렬 (1순위)
- 이렇게 하면 길이가 같은 단어들은 1단계의 사전순을 유지하면서 정렬됩니다.

### 3. sort()의 key 매개변수
- `lst.sort(key=len)`처럼 `key` 매개변수를 사용하면 정렬 기준을 지정할 수 있습니다.
- `len` 함수를 전달하면 각 요소의 길이를 기준으로 정렬됩니다.
- 괄호 없이 `len`만 전달하는 것이 포인트입니다.

### 4. sort()의 반환값
- `sort()` 메서드는 리스트를 정렬하지만 **None을 반환**합니다.
- 따라서 `print(lst.sort())`를 하면 `None`이 출력됩니다.
- 정렬 후 출력하려면 `lst.sort()` 실행 후 따로 `print(lst)`를 해야 합니다.

### 5. 리스트 순회 출력
- `print(lst)`는 리스트 전체를 `['a', 'b', 'c']` 형태로 출력합니다.
- 각 요소를 한 줄씩 출력하려면 `for` 문으로 순회하며 출력해야 합니다.
