# 최빈값 찾기

## 문제 설명 
일차원 정수 배열 numbers가 주어질 때 numbers의 최빈값을 반환하는 solution 함수를 작성하시오.

만약 numbers에 가장 많은 등장 빈도를 갖는 숫자가 여러 개라면 그중에서 크기가 가장 작은 수를 반환하시오.

## 제한 사항
numbers의 길이 n은 1 이상 100,000 이하의 자연수입니다.
numbers의 각 원소는 -1,000 이상 1,000 이하의 정수입니다.

### dailyalgo 171번 문제

```python
def solution(numbers):
    freq = {}

    # 1. 각 숫자의 등장 횟수 세기
    for num in numbers:
        if num in freq:
            freq[num] += 1
        else:
            freq[num] = 1

    # 2. 최빈값의 등장 횟수 구하기
    max_freq = max(freq.values())

    # 3. 최빈값 후보들을 모으기
    candidates = []
    for num, count in freq.items():
        if count == max_freq:
            candidates.append(num)

    # 4. 후보 중 가장 작은 값 반환
    answer = min(candidates)
    return answer
```
```python
# 2와 # 3을 한 줄 코드로 변경한 버전
def solution(numbers):
    freq = {}

    for num in numbers:
        freq[num] = freq.get(num, 0) + 1

    max_freq = max(freq.values())

    return min(num for num, count in freq.items() if count == max_freq)

```
